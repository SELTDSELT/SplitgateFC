# Splitgate Friend Code Miner
For some reason, viruses really like mining for crypto, why not mine for Splitgate Friend Codes?

Created by NotDaBenjamin, this absolute dogshit project generates Splitgate friend codes that may or may not work.

### How to use
**Make sure you have Python installed!!**
1. On windows, open file explorer and go to the folder with all the files.
2. While still in the folder, type in the bar at the top "cmd" and then press enter.
3. Type in the command prompt window "py main.py" or "py splitgamifymycommandprompt.py"
4. Type !!help to see a list of commands.
5. Type !!pull to generate 1 random friend code.
6. Type !!mine and follow the instructions to mine 2 friend codes every second!

TIP: You can also use !!uncappull or !!uncapmine to get friend codes without capitalization, making it easier to read.
TIP x2: If you want to get straight to mining you can just type "py mine/script.py"!
